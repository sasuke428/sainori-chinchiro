# さいのり専用チンチロ

GitHub Pages 公開用の静的Webアプリです。

## 公開手順

1. GitHubで新規リポジトリを作成
2. このZIPを解凍
3. `index.html`, `style.css`, `script.js`, `.nojekyll` をリポジトリ直下にアップロード
4. Settings → Pages → Build and deployment
5. Source: Deploy from a branch
6. Branch: main / root を選択して Save

数十秒〜数分後に `https://ユーザー名.github.io/リポジトリ名/` で公開されます。
